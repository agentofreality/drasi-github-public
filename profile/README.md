**Drasi** is a [CNCF Sandbox](https://www.cncf.io/projects/drasi/) project for real-time evaluation of changes in data and automating meaningful reactions.

## Get Started

- 📖 [Documentation](https://drasi.io)
- 💬 [Discord Community](https://aka.ms/drasidiscord)
- ⚙️ [drasi-core](https://github.com/drasi-project/drasi-core) - The core rust crates that power Drasi. Embeddable in any rust process.
- 💻 [drasi-server](https://github.com/drasi-project/drasi-platform) - Drasi as a standalone process
- 🐙 [drasi-platform](https://github.com/drasi-project/drasi-platform) - Drasi for Kubernetes
- 📄 [docs](https://github.com/drasi-project/docs) - Powers the [drasi website](drasi.io)
- 🎭 [learning](https://github.com/drasi-project/learning) - Tutorials and Sample code to learn about Drasi.
