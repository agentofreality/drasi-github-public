# Support

Looking for help with Drasi? Here's how to get support.

## Documentation

Visit our documentation site for guides, tutorials, and reference material:

- [https://drasi.io](https://drasi.io)

## Community Chat

Join our Discord server to chat with maintainers and community members:

- [https://aka.ms/drasidiscord](https://aka.ms/drasidiscord)

## Reporting Issues

Found a bug or have a feature request? Open an issue in the relevant repository:

- [drasi-platform](https://github.com/drasi-project/drasi-platform/issues)
- [drasi-core](https://github.com/drasi-project/drasi-core/issues)
- [drasi-server](https://github.com/drasi-project/drasi-server/issues)
- [docs](https://github.com/drasi-project/docs/issues)

## Security Issues

Please report security vulnerabilities privately via email — **not** through public GitHub issues.

- [security@drasi.io](mailto:security@drasi.io)

See [SECURITY.md](https://github.com/drasi-project/.github/blob/main/SECURITY.md) for details.

## General Contact

- [codeofconduct@drasi.io](mailto:codeofconduct@drasi.io) — Code of Conduct inquiries
